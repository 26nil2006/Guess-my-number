# Guess My Number

**Guess My Number** is a simple number guessing game built using HTML, CSS, and JavaScript. The objective of the game is for the player to guess a secret number between 1 and 20. The game provides feedback on whether the guessed number is too high, too low, or correct.

## Features

- **Range of Numbers:** The secret number is randomly generated within the range of 1 to 20.
- **Feedback:** Players receive immediate feedback after each guessâ€”whether the guess is too high, too low, or correct.
- **Score Tracking:** The player's score starts at 20 and decreases by 1 for each incorrect guess.
- **Highscore:** The game tracks the highest score achieved in a session.

## How to Play

1. **Start the Game:** When the game loads, a secret number between 1 and 20 is randomly selected.
2. **Make a Guess:** Enter a number between 1 and 20 in the input field and click the "Check!" button.
3. **Receive Feedback:** The game will display if your guess was too high, too low, or correct.
4. **Score:** If your guess is incorrect, your score decreases by 1. If you guess the correct number, your remaining score will be displayed as the final score.
5. **Play Again:** Click the "Again!" button to reset the game with a new secret number and try again.

## Technologies Used

- **HTML:** Structure of the web page.
- **CSS:** Styling and layout of the game.
- **JavaScript:** Game logic, including number generation, feedback, and score tracking.
